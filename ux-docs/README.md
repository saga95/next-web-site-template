# UX Docs

Output folder for **Aria**, the UX Designer Agent (see [`../CLAUDE.md`](../CLAUDE.md)).

## Structure

| Folder                      | Contents                                   | Generated by   |
| --------------------------- | ------------------------------------------ | -------------- |
| `personas/`                 | User persona markdown files                | `/persona`     |
| `empathy-maps/`             | Empathy map markdown files                 | `/empathy-map` |
| `information-architecture/` | IA diagrams (Mermaid) + notes              | `/ia`          |
| `sitemaps/`                 | Sitemap diagrams (Mermaid) + `sitemap.xml` | `/sitemap`     |
| `user-flows/`               | User flow diagrams (Mermaid)               | `/userflow`    |
| `user-journeys/`            | Journey maps with emotion curves           | `/journey`     |
| `wireframes/`               | Lo-fi React wireframe components           | `/wireframe`   |

All artifacts must reference design tokens from
[`../design-system/tokens.ts`](../design-system/tokens.ts). See
[`../.claude/commands/`](../.claude/commands/) for full command specifications.
